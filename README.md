# Subsea Inspection Report Generator

Python tool for generating automated reports from subsea inspection logs.

This project demonstrates how offshore inspection data can be processed using Python to produce structured operational summaries.

---

## Features

- Parse subsea inspection logs
- Generate inspection summaries
- Count inspected assets
- Analyze inspection results

---

## Technologies

Python
Pandas
